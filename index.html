Project 1: Portfello (Light Theme)

This version of Portfello adopts a clean and elegant light theme aimed at delivering a professional and inviting feel. The design emphasizes simplicity, with soft colors and spacious layout elements, ideal for a modern portfolio. Advanced animations created with HTML, CSS, and JavaScript enhance the user experience by adding smooth transitions and effects that bring the interface to life. The project leverages Font Awesome icons and custom Google Fonts to add personality and clarity, making this portfolio both visually appealing and highly functional.

Project 2: Portfello (Dark Theme)

The second iteration of Portfello explores a dark theme, catering to a more contemporary and bold aesthetic. With rich contrasts and high-impact visuals, this design emphasizes a sleek, modern look, making it perfect for tech-savvy users. The dark theme, combined with dynamic animations, provides an immersive browsing experience powered by HTML, CSS, and JavaScript. Font Awesome icons and carefully selected Google Fonts further enhance the edgy style, showcasing creativity and technical skill while maintaining excellent usability.
